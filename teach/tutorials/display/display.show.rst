::::::::::::::::::

	动手DIY一颗跳动的心。

	拼插源码

	.. image:: images/display/display.shows.png

	实例源码::

		from microbit import *

		while True:
		  display.show(Image.HEART)
		  sleep(500)
		  display.show(Image.HEART_SMALL)
		  sleep(500)
